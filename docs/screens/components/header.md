# 共通ヘッダー

## 概要

全画面で共通して使用するヘッダーコンポーネントです。ログイン状態に応じて表示内容が変化します。

## レイアウト

### 未ログイン時

```
+----------------------------------+
| タイトル        [ログイン][登録] |
+----------------------------------+
```

### ログイン済み時

```
+----------------------------------+
| タイトル      表示名 [ログアウト] |
+----------------------------------+
```

## 表示項目

| 項目           | データ型 | 説明                         | 表示条件     |
| -------------- | -------- | ---------------------------- | ------------ |
| タイトル       | string   | 画面タイトル（画面ごとに異なる） | 常に表示     |
| ユーザー表示名 | string   | ログイン中のユーザーの表示名 | ログイン済み |

## 操作項目

| 要素             | 種類   | 動作                   | 表示条件     |
| ---------------- | ------ | ---------------------- | ------------ |
| ログインボタン   | ボタン | ログイン画面へ遷移     | 未ログイン   |
| 登録ボタン       | ボタン | サインアップ画面へ遷移 | 未ログイン   |
| ログアウトボタン | ボタン | ログアウト処理実行     | ログイン済み |

## 画面遷移

| 遷移先                     | トリガー                     |
| -------------------------- | ---------------------------- |
| ログイン (SC-AUTH-001)     | 「ログイン」ボタンクリック   |
| サインアップ (SC-AUTH-002) | 「登録」ボタンクリック       |
| タイムライン (SC-POST-001) | 「ログアウト」ボタンクリック |

## 備考

- ログアウト後はタイムライン画面へリダイレクト
- 認証画面（ログイン、サインアップ）では右上のボタンを非表示
  - ヘッダー自体は表示されるが、ログイン/登録/ログアウトボタンは表示されない
  - タイトルのみ表示される
