# ログイン

## 基本情報

| 項目       | 内容                             |
| ---------- | -------------------------------- |
| 画面 ID    | SC-AUTH-001                      |
| 画面名     | ログイン                         |
| URL/ルート | `/login`                         |
| 説明       | ユーザー名とパスワードでログイン |

## 概要

ユーザー名とパスワードを入力してログインします。認証成功時、タイムラインへ遷移します。

## レイアウト

```
+----------------------------------+
| ログイン                         |
+----------------------------------+
| ユーザー名                       |
| +------------------------------+ |
| | テキスト入力                 | |
| +------------------------------+ |
|                                  |
| パスワード                       |
| +------------------------------+ |
| | パスワード入力               | |
| +------------------------------+ |
|                                  |
|              [ログイン]          |
|                                  |
| アカウントをお持ちでない方は     |
| [新規登録]                       |
+----------------------------------+
```

## 入力項目

| 項目       | 種類     | 必須 | 説明                     |
| ---------- | -------- | ---- | ------------------------ |
| ユーザー名 | text     | ○    | ログイン ID              |
| パスワード | password | ○    | パスワード（マスク表示） |

## 操作項目

| 要素     | 種類   | 動作                                              |
| -------- | ------ | ------------------------------------------------- |
| ログイン | ボタン | 認証 API を呼び出し、成功したらタイムラインへ遷移 |
| 新規登録 | リンク | サインアップ画面へ遷移                            |

## API 連携

| 項目           | 内容                         |
| -------------- | ---------------------------- |
| エンドポイント | `POST /auth/login`           |
| タイミング     | 「ログイン」ボタンクリック時 |

詳細は [open-api.yaml](../../open-api.yaml) を参照

## 画面遷移

| 遷移先                     | トリガー                   |
| -------------------------- | -------------------------- |
| タイムライン (SC-POST-001) | ログイン成功時             |
| サインアップ (SC-AUTH-002) | 「新規登録」リンククリック |

## バリデーション

| 項目       | ルール | エラーメッセージ             |
| ---------- | ------ | ---------------------------- |
| ユーザー名 | 必須   | ユーザー名を入力してください |
| パスワード | 必須   | パスワードを入力してください |

## エラーハンドリング

| エラー         | 表示内容                                       |
| -------------- | ---------------------------------------------- |
| 401 (認証失敗) | 「ユーザー名またはパスワードが間違っています」 |
| API 送信失敗   | 「ログインに失敗しました」                     |

## 備考

- 既にログイン済みの場合はタイムラインへリダイレクト
