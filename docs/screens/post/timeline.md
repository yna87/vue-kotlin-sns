# タイムライン

## 基本情報

| 項目       | 内容                           |
| ---------- | ------------------------------ |
| 画面 ID    | SC-POST-001                    |
| 画面名     | タイムライン                   |
| URL/ルート | `/`                            |
| 説明       | 投稿の一覧を表示するメイン画面 |

## 概要

全ての投稿を新しい順に一覧表示します。ユーザーは投稿を閲覧し、投稿作成画面へ遷移できます。

## レイアウト

```
+----------------------------------+
| タイムライン        [投稿する]   |
+----------------------------------+
| 投稿1                            |
| 投稿本文テキスト...              |
| 2026-01-11 10:30                 |
+----------------------------------+
| 投稿2                            |
| 投稿本文テキスト...              |
| 2026-01-11 09:15                 |
+----------------------------------+
| ...                              |
+----------------------------------+
```

## 表示項目

各投稿カードに以下を表示：

| 項目     | データ型 | 説明               |
| -------- | -------- | ------------------ |
| 投稿本文 | string   | テキストコンテンツ |
| 投稿日時 | string   | 投稿日時           |

## 操作項目

| 要素           | 種類   | 動作               |
| -------------- | ------ | ------------------ |
| 投稿するボタン | ボタン | 投稿作成画面へ遷移 |

## API 連携

| 項目           | 内容             |
| -------------- | ---------------- |
| エンドポイント | `GET /posts` |
| タイミング     | 画面表示時       |

詳細は [open-api.yaml](../../open-api.yaml) を参照

## 画面遷移

| 遷移先                     | トリガー                   |
| -------------------------- | -------------------------- |
| 投稿作成画面 (SC-POST-002) | 「投稿する」ボタンクリック |

## エラーハンドリング

| エラー       | 表示内容                     |
| ------------ | ---------------------------- |
| API 取得失敗 | 「投稿の取得に失敗しました」 |

## 備考

- 投稿は新しい順（createdAt の降順）で表示
- 初期表示では全投稿を取得
