# タイムライン

## 基本情報

| 項目       | 内容                           |
| ---------- | ------------------------------ |
| 画面 ID    | SC-POST-001                    |
| 画面名     | タイムライン                   |
| URL/ルート | `/`                            |
| 説明       | 投稿の一覧を表示するメイン画面 |

## 概要

全ての投稿を新しい順に一覧表示します。未ログインユーザーも投稿を閲覧できます。ログイン済みユーザーは投稿作成画面へ遷移できます。

## レイアウト

ヘッダーは [共通ヘッダー](../components/header.md) を参照

### 未ログイン時

```
+----------------------------------+
| [共通ヘッダー]                   |
+----------------------------------+
| 表示名                           |
| @user_name                       |
| 投稿本文テキスト...              |
| 2026-01-11 10:30                 |
+----------------------------------+
| 表示名2                          |
| @user_name2                      |
| 投稿本文テキスト...              |
| 2026-01-11 09:15                 |
+----------------------------------+
| ...                              |
+----------------------------------+
```

### ログイン済み時

```
+----------------------------------+
| [共通ヘッダー]                   |
+----------------------------------+
|              [投稿する]          |
+----------------------------------+
| 表示名                           |
| @user_name                       |
| 投稿本文テキスト...              |
| 2026-01-11 10:30                 |
+----------------------------------+
| 表示名2                          |
| @user_name2                      |
| 投稿本文テキスト...              |
| 2026-01-11 09:15                 |
+----------------------------------+
| ...                              |
+----------------------------------+
```

## 表示項目

### 各投稿カード

| 項目           | データ型 | 説明                   |
| -------------- | -------- | ---------------------- |
| ユーザー名     | string   | 投稿者のユーザー名     |
| 投稿者表示名   | string   | 投稿者の表示名         |
| 投稿本文       | string   | テキストコンテンツ     |
| 投稿日時       | string   | 投稿日時               |

## 操作項目

| 要素           | 種類   | 動作                   | 表示条件     |
| -------------- | ------ | ---------------------- | ------------ |
| 投稿するボタン | ボタン | 投稿作成画面へ遷移     | ログイン済み |

## API 連携

| 項目           | 内容             |
| -------------- | ---------------- |
| エンドポイント | `GET /posts` |
| タイミング     | 画面表示時       |

詳細は [open-api.yaml](../../open-api.yaml) を参照

## 画面遷移

| 遷移先                     | トリガー                   |
| -------------------------- | -------------------------- |
| 投稿作成画面 (SC-POST-002) | 「投稿する」ボタンクリック |

※ヘッダーからの画面遷移は [共通ヘッダー](../components/header.md) を参照

## エラーハンドリング

| エラー       | 表示内容                     |
| ------------ | ---------------------------- |
| API 取得失敗 | 「投稿の取得に失敗しました」 |

## 備考

- 投稿は新しい順（createdAt の降順）で表示
- 初期表示では全投稿を取得
- 未ログインユーザーも投稿の閲覧は可能
